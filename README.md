## これは何？

WordPress サイトにカスタマイズ可能なローディング画面を追加するプラグインです。サイトアクセス時だけでなく、サイト内ページ遷移時にも表示されます。

### 主な機能

- オリジナルローディング画像をメディアライブラリから選択
- 2つのアニメーションタイプ（ワイプ演出/プログレスバー）
- 4方向のワイプ（下から上/上から下/左から右/右から左）
- プログレスバー表示
- カスタマイズ可能な背景（色・画像・ぼかし効果）
- 表示頻度の選択（初回のみ/毎回表示）
- CarryPod連携（設定保存時に自動キャッシュクリア）
- プレビュー機能（保存前に確認可能）

---

## インストール

1. [Releases](https://github.com/villyoshioka/Screw/releases) から ZIP ファイルをダウンロード
2. WordPress の管理画面で「プラグイン」→「新規追加」→「プラグインのアップロード」
3. ダウンロードした ZIP ファイルを選択してインストール
4. 「有効化」をクリック

---

## 使い方

プラグインを有効化すると、WordPress 管理画面に「Screw」メニューが追加されます。

### 基本設定

1. **ローディング画像を選択**（必須）
   - メディアライブラリから任意の画像を選択
   - 横幅を指定（デフォルト: 90px）

2. **アニメーションタイプを選択**
   - **ワイプ演出**: 下から上/上から下/左から右/右から左
   - **プログレスバー**: リアルタイムでロード状況を表示

3. **背景を設定**
   - 背景色を選択（デフォルト: 白）
   - 背景画像を選択（オプション）
   - 背景画像をぼかす（オプション）

4. **表示頻度を選択**
   - **初回のみ**: サイト訪問時の最初のページのみ表示
   - **毎回表示**: サイト内ページ遷移時も毎回表示

5. 「設定を保存」ボタンをクリック

### プレビュー機能

設定画面の「プレビュー」ボタンをクリックすると、保存前に現在の設定でローディング画面を確認できます（別タブで開きます）。

### 設定のリセット

「設定をリセット」ボタンをクリックすると、全ての設定がデフォルト値に戻ります。

---

## CarryPod連携

CarryPod プラグインがインストールされている場合、Screw の設定を保存すると自動的にCarryPod のキャッシュがクリアされます。

## プライバシーについて

このプラグインはローディング画面を表示する機能のみを提供します。

- ユーザーデータの収集・解析なし
- トラッキング機能なし
